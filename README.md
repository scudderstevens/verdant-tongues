## Packages

**SETUP** 

Yarn recipe (TypeScript & PnP) [docs](https://yarnpkg.com/getting-started/recipes#typescript--pnp-quick-start)

`yarn init -2`

`yarn add --dev typescript`

`yarn dlx @yarnpkg/pnpify --sdk vscode`

`yarn plugin import typescript`
